Water Quality: Description back to ToC

This vocabulary extends the W3C Semantic Sensor Network Ontology (known as SOSA) with a set of properties that are specific to the water quality domain, with the goal of representing water quality data measured by water quality stations from homogeneous shape. The ontology that extends with this vocabulary, W3C Semantic Sensor Ontology (SOSA), is represented in the following figure, where you can see how the different measures that are taken, represented by sosa:Observation, and its value, sosa:Result, of a specific property, sosa:ObservableProperty, of an object concrete, sosa:featureOfInterest, are measured by a sensor sosa :Sensor which, in turn, is contained by the stations that perform the measurements, sosa:Platform.


In this case, the ontology is expanded to represent the classes that are specific to the domain to be represented, in this case water quality, as shown in the following figure:
Additional classes of this vocabulary are:
  • calidad-agua:WaterQualityStation: It refers to the stations that measure the properties of the water, on which its quality is determined.
  • calidad-agua:WaterQualityObservation: It refers to each of the different measures that are taken from dbpedia:Water.
  • calidad-agua:WaterQualityProperty: It refers to the different properties about the water on which measurements are taken.
  • calidad-agua:WaterQualitySensor: It refers to each of the specific sensors for measuring the properties of water found in the measurement stations.
